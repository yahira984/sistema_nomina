Coloca aqui la credencial JSON de Firebase Admin SDK.

Nombre recomendado:
service-account.json

En el archivo .env configura:
FIREBASE_CREDENTIALS=app/firebase/service-account.json
FIREBASE_DATABASE_URL=https://TU-PROYECTO-default-rtdb.firebaseio.com

No subas el JSON al repositorio ni lo compartas por WhatsApp/correo.
